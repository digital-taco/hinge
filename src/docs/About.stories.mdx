# Hinge UI

This library acts as a _base_ library for any other component library. It's set of components are composed of common components found in almost every component library available today, but _without styling_. It's left to the consumer to build their own theming system on top of Hinge UI.

Github Project: https://github.com/orgs/digital-taco/projects/1/views/1
